# About The Project

This project is a poll collecting webiste collecting the Name , Caption , url time and storing it in database.
It allows to edit , deleting the memes.


### Built With

* [Bootstrap](https://getbootstrap.com)
* [JQuery](https://jquery.com)
* [MongoDB](https://cloud.mongodb.com/)

# Getting Started with npm init

This project was bootstrapped with [npm init](https://github.com/song940/npm-init).

### `npm start`

### Installation

1. Clone the repo
   ```sh
   git clone https://gitlab.crio.do/COHORT_ME_BUILDOUT_XMEME_ENROLL_1612436694845/sunidhihegde1999-me_buildout_xmeme
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Enter your MONGODBURL in `config.js`
   ```JS
   MONGODBURL: 'mongodb://localhost:27017/YOUR_DATABASE_NAME_HERE' 
   ```
4. Run npm start 
    ```JS
   E:/YOUR_FILENAME/:  npm start
   ```
